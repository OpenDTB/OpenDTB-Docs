zjb.main.analysis.mne_process.CreateMNESignals#

class zjb.main.analysis.mne_process.CreateMNESignals(**traits)[源代码]#

从仿真的脑区神经活动创建MNE格式的电生理信号

fwd#

mne正向解实例

Type:

Forward

raw#

mne的传感器空间数据实例，包含mne的采集信息info

Type:

Raw

mapping#

皮层-脑区映射实例

Type:

SurfaceRegionMapping

regional_ts#

脑区时间序列实例

Type:

RegionalTimeSeries

src#

可以是源空间文件的路径，或是SourceSpaces实例

Type:

path-like | SourceSpaces实例

bem#

边界元模型，表明头模型每一层的导电率

Type:

path-like | ConductorModel实例

trans#

头模型 head<->MRI 转换的文件.

Type:

str | None

eeg#

表明是否输出eeg的数据

Type:

Bool

meg#

表明是否输出meg的数据

Type:

Bool

_build_fwd()[源代码]#

如果没有fwd，则通过src, trans和bem生成一个fwd

_default_raw()[源代码]#

如果没有raw，则生成默认的raw
